# Workflow preview images

Upload the generated preview images here using this exact naming format:

- wf-001.jpg
- wf-002.jpg
- wf-003.jpg
- ...
- wf-050.jpg

The website cards automatically look for images at:

/public/workflow-previews/{workflow-id-lowercase}.jpg

Example:
WF-001 uses /workflow-previews/wf-001.jpg
